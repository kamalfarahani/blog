# Zen Theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the plain grey stylesheet with a warm parchment-textured zen theme using Lora serif, cream tones, and ink-style details.

**Architecture:** Three file changes — `base.html` gains a Google Fonts link and a `.content-card` wrapper div; `post_detail.html` replaces an inline style with a CSS class on the featured image; `style.css` is fully replaced with the zen theme. No model, view, or URL changes.

**Tech Stack:** CSS, Django templates, Google Fonts (Lora)

---

## File Map

| File | Action | Purpose |
|------|--------|---------|
| `posts/templates/posts/base.html` | Modify | Add Lora font links, wrap body content in `.content-card`, remove `<hr />` |
| `posts/templates/posts/post_detail.html` | Modify | Replace inline image style with `class="post-image"` |
| `posts/static/posts/style.css` | Replace | Full zen theme |

## Task 1: Update `base.html`

**Files:**
- Modify: `posts/templates/posts/base.html`

The `.content-card` wrapper is what creates the "paper on textured background" effect — without it, the card shadow and background have nothing to attach to. The `<hr />` between nav and content is removed because the nav will get a CSS bottom border instead.

- [ ] **Step 1: Replace the full contents of `posts/templates/posts/base.html`**

```html
{% load static %}
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>{% block title %}Blog{% endblock %}</title>
    <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link href="https://fonts.googleapis.com/css2?family=Lora:ital,wght@0,400;0,700;1,400&display=swap" rel="stylesheet" />
    <link rel="stylesheet" href="{% static 'posts/style.css' %}" />
  </head>
  <body>
    <div class="content-card">
      <nav><a href="{% url 'post_list' %}">Blog</a></nav>
      {% block content %}{% endblock %}
    </div>
  </body>
</html>
```

## Task 2: Update `post_detail.html`

**Files:**
- Modify: `posts/templates/posts/post_detail.html`

The featured image currently has an inline style (`style="max-width: 100%; margin-bottom: 1rem"`). Replace it with `class="post-image"` so the zen CSS can apply the full image treatment (border-radius, warm bottom border, full width).

Also fix the linter-broken template tag on line 1 (extends + block title on same line) and the split `{% if post.category %}` across lines 6–7 — Django's template parser requires each `{% tag %}` to be on a single line.

- [ ] **Step 1: Replace the full contents of `posts/templates/posts/post_detail.html`**

```html
{% extends 'posts/base.html' %}

{% block title %}{{ post.title }}{% endblock %}

{% block content %}
<article>
  <h1>{{ post.title }}</h1>
  <p class="post-meta">
    {{ post.published_at|date:"F j, Y" }}
    {% if post.category %} &middot; {{ post.category }}{% endif %}
  </p>
  {% if post.featured_image %}
  <img class="post-image" src="{{ post.featured_image.url }}" alt="{{ post.title }}" />
  {% endif %}
  <div class="post-body">{{ post.body|linebreaks }}</div>
  {% if post.tags.all %}
  <div class="tags">
    {% for tag in post.tags.all %}<span>{{ tag }}</span>{% endfor %}
  </div>
  {% endif %}
</article>
<p class="back-link"><a href="{% url 'post_list' %}">&larr; Back to all posts</a></p>
{% endblock %}
```

## Task 3: Replace `style.css` with zen theme

**Files:**
- Modify: `posts/static/posts/style.css`

This is the full theme. The SVG data URI uses `feTurbulence` noise at `0.04` opacity for paper grain. The `#` in the SVG's internal `url(#n)` reference is encoded as `%23` for CSS compatibility.

- [ ] **Step 1: Replace the full contents of `posts/static/posts/style.css`**

```css
/* ── Reset ──────────────────────────────────────────────── */
*, *::before, *::after {
  box-sizing: border-box;
}

/* ── Page & texture ─────────────────────────────────────── */
body {
  font-family: 'Lora', Georgia, serif;
  font-size: 1.125rem;
  line-height: 1.9;
  color: #2c1f0e;
  background-color: #f2ebe0;
  background-image: url("data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' width='200' height='200'><filter id='n'><feTurbulence type='fractalNoise' baseFrequency='0.75' numOctaves='4' stitchTiles='stitch'/><feColorMatrix type='saturate' values='0'/></filter><rect width='200' height='200' filter='url(%23n)' opacity='0.04'/></svg>");
  margin: 0;
  padding: 2rem 1rem;
}

/* ── Content card ───────────────────────────────────────── */
.content-card {
  max-width: 660px;
  margin: 0 auto;
  background: #faf7f2;
  padding: 2.5rem 3rem;
  border-radius: 4px;
```
